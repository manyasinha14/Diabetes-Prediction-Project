# Diabetes Prediction using Machine Learning
This project aims to predict the likelihood of diabetes in patients using supervised machine learning techniques. The analysis is performed on a medical dataset containing relevant health attributes, and multiple models are trained and evaluated to achieve optimal performance.

## Project Overview
The notebook covers the complete machine learning pipeline, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation. Ensemble techniques are also applied to enhance prediction accuracy.

## Models Used
- XGBoost
- LightGBM
- CatBoost

## Evaluation
Model performance is evaluated using ROC-AUC and other classification metrics. Probability and rank blending techniques are used to improve overall results.

## Tools & Technologies
- Python
- Jupyter Notebook
- Scikit-learn
- XGBoost, LightGBM, CatBoost

## Dataset
The dataset used for this project is sourced from Kaggle.

## Conclusion
This project demonstrates the application of machine learning algorithms to a real-world healthcare problem, highlighting model comparison and ensemble learning for improved prediction performance.
